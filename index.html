Editar objeto

Clique e arraste qualquer caixa para mover. Também é possível mover com as setas do teclado. Use Shift + seta para mover mais rápido. Use o ponto vermelho no canto para aumentar ou diminuir. Para excluir, selecione o objeto e clique em Excluir objeto selecionado. Depois clique em Salvar alterações para gravar tudo no Firebase.
No celular, arraste a tela para navegar. No modo admin, toque em uma caixa para editar.
